# Titanic-survival-predict

This project develops a Machine Learning classification model to predict diabetes outcomes using patient health data. The workflow includes data loading, exploratory data analysis, feature visualization, data preprocessing, train-test splitting, and feature scaling using StandardScaler. A Random Forest Classifier is trained to identify patterns associated with diabetes and generate predictions on unseen data. The model's performance is evaluated using a confusion matrix and classification report, providing insights into accuracy, precision, recall, and overall classification effectiveness. This project demonstrates an end-to-end machine learning pipeline from data preparation and model development to evaluation and interpretation of results.
